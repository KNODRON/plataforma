# Plataforma de Análisis Criminal Estratégico - Maqueta OS.9

Maqueta funcional estática para GitHub Pages.

## Incluye
- Login simulado.
- Dashboard por módulos.
- Formulario funcional de Secuestro basado en la matriz Excel.
- Guardado local en el navegador (`localStorage`).
- Exportación JSON del caso actual.

## Importante
Esta versión no tiene base de datos ni login real. No ingresar datos sensibles reales. La etapa siguiente puede migrarse a Python/Flask para usuarios, base de datos, reportes y red interna.
